# AnimalAPP

Random pictures of dogs or cats.

![](/foto.png)

## Configuration

To choose whether to show dog or cat pictures, set the `ANIMAL` env var to `dog` or `cat`.

To change the language of the app to spanish, create a `appconf.json` with the following content, in the same directory as the app:
```
{
    "idioma": "espa√±ol"
}
```
